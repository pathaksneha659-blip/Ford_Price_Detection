# Ford_Price_Detection
Ford Price Prediction
🚗 Ford Price Prediction Model
📌 Project Overview

This project is a machine learning demo that predicts the price of Ford cars based on various influencing factors such as features, specifications, and encoded categorical variables.

Two different encoding approaches were tested to improve model performance:

Label Encoding
One-Hot Encoding

The results show that encoding strategy plays an important role in model accuracy.

🎯 Objective

To build a regression model that can predict Ford car prices using machine learning techniques and compare the impact of different encoding methods on model performance.

🧠 Techniques Used
Data Preprocessing
Feature Encoding:
Label Encoding
One-Hot Encoding
Machine Learning Model (Regression)
Train-Test Split
Model Evaluation (Accuracy Score / R² Score)
📊 Results
Encoding Method	Accuracy Score
Label Encoding	73%
One-Hot Encoding	75%

👉 One-Hot Encoding performed slightly better, showing that proper handling of categorical variables improves model performance.
